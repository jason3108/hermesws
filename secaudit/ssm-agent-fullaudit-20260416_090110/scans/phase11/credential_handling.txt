=== Phase 11: Credential Handling 0day挖掘 ===

## Target: AWS Credentials & EC2 Metadata

### Code Locations
- agent/credentials/
- agent/ec2metadata/

### Potential Vulnerabilities

1. **EC2 Metadata Service SSRF**
   - Issue: Agent may access arbitrary metadata endpoints
   - Impact: Credential theft via SSRF
   
2. **Credential Caching**
   - Issue: Credentials cached in memory longer than necessary
   - Impact: Credential exposure in memory dumps

3. **IMDSv1 Support**
   - Issue: May fall back to IMDSv1 (no token required)
   - Impact: Easier SSRF exploitation

## Finding: HIGH - IMDSv2 enforcement needed
